this is a dive that makes an example of attribute selectors

Remember, you can use attribute selectors like this "div[id]{color: blue}" and it makes all the divs with an id looking the same in css


What is more if you want all the ids within a div that part of the names is the same you can use div[id~="xxx"]{color: green;}" then all the id within a div that name is similar to xxx will have the same look


OR! use "div[id^="xxx"]{...} then everything that begins with xxx will look the same


OR! use "div[id$="xxx"]{...} then everything that ends with xxx will look the same
